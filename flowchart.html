Scrape Reddit
reddit-scraper.js → Apify API
Automated
▼

Input

  • Reddit URLs (subreddits + search queries)
  • Topic name, sort order, date filter
  • Config: scrape-config.json

Process

  • Call Apify trudax~reddit-scraper-lite
  • Poll until scrape completes (1-5 min)
  • Deduplicate against .seen-urls.json
  • Filter by date if --since specified

Output

  • {topic}/Scrapes/scrape-{ts}.json
  • Posts: title, author, body, score, URL
  • Comments: author, body, subreddit, URL
  • Logged to scrape-history.jsonl
Cost: ~$0.003-0.005 / result Features: parallel URLs, cross-run dedup, date filtering

Filter & Pre-process
lead-scorer.js → Local processing
Automated
▼

Filters Applied

  • Remove bots (automoderator, [deleted])
  • Remove career seekers (2+ phrase matches)
  • Remove deleted/removed posts
  • Require min 30 chars text length

Smart Override

  • Buyer signal detection keeps high-intent posts
  • Budget mentions, tool evaluations, vendor comparisons
  • Overrides career-seeker filter if buying intent found

Output

  • {topic}/pending-leads.txt
  • Formatted: [index] TYPE | AUTHOR | SUB | BODY
  • Pre-filter summary report

Scoring Criteria

  • HOT (8-10): Decision-maker, actively buying, company identified
  • WARM (5-7): Practitioner with pain points, not actively buying
  • COLD (1-4): Mentioned topic but unlikely buyer

AI Generates

  • Score 1-10 per lead
  • Tier assignment (HOT / WARM / COLD)
  • Reasoning for the score
  • Personalized outreach message draft

Human Review & Prioritize
Review leads-{date}.md report
Manual

Review Process

  • Read the markdown report with scored leads
  • Identify highest-value HOT leads
  • Validate AI scoring against your judgement
  • Plan outreach strategy and timing

Decision Point

  • Which leads to enrich further?
  • Select tiers to proceed: --tiers HOT,WARM
  • Cap enrichment: --max-users N
  • Choose research depth (with/without Perplexity)
5
Enrich & Research Prospects
enrich-leads.js → Apify + Perplexity API
Automated
▼

Step A: Profile Scraping

  • Extract usernames from leads JSON
  • Scrape full Reddit history via Apify
  • Cache in Profiles/{user}.json
  • Up to 100 posts/comments per user

Step B: Signal Extraction

  • Company hints ("at my org", "we use")
  • Role hints ("I manage", "as an engineer")
  • Location hints (cities, regions)
  • Tech stack (Okta, Entra, CyberArk...)

Step C: Perplexity Research

  • Query Perplexity Sonar API with signals
  • Find LinkedIn profile URL
  • Identify real name & employer
  • Extract email if available
↓

Final Output

  • Enriched/enriched-{date}.json + .md
  • Revised scores (vendor accounts downgraded)
  • Company, role, location, tech stack
  • LinkedIn URL, email, enhanced outreach message
Cost: ~$0.30 Apify/user ~$0.05 Perplexity/user 3 modes: scrape-only, full research, research-only (cached)

Bonus Tools

📅
Scheduler

schedule.js — Runs all scrape jobs weekly via macOS launchd. Install with --install, runs every Monday at 8 AM. Auto-generates leads across all topics.

📈
Trend Analysis

trends.js — Analyzes volume over time, extracts top keywords, identifies rising topics, and finds recurring authors across all scrape history.

🔍
Subreddit Discovery

discover-subreddits.js — Scans existing scrape data for mentioned subreddits and cross-posting patterns to recommend new communities to monitor.
